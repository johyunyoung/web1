파이썬

1991년에 발표된 인터프리터 방식[2]의 프로그래밍 언어.

창시자는 귀도 반 로섬(Guido van Rossum). 네덜란드어 발음으로 '히도 판 로쉼'[3] 1989년 크리스마스 주에, 연구실이 닫혀있어서 심심한 김에 만들었다. 농담이 아니고 반 로섬을 유럽에서는 애덤 스미스에 비견할 정도다. 프로그래밍 계의 경제학자라나... 심심해서 만들었다는 것은 파이썬 서문과 마이크로소프트웨어와 한 인터뷰를 보면 알겠지만 사실이다. 기술자들은 심심할 때 항상 걸작이 나온다. 2000년에는 Python 2, 2008년에는 Python 3가 나왔다.
